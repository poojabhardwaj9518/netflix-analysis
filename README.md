# Netflix Movies and TV Shows Analysis

A data analysis project focused on exploring and querying Netflix's catalog of movies and TV shows. This project uses SQL to clean, segment, and extract insights from the dataset, answering key business questions about content distribution, ratings, and release trends.

## Repository Structure
  **netflix_titles.xlsx**: The raw dataset containing metadata for movies and TV shows available on Netflix.
*   **NETFLIX.sql**: The core script containing the SQL queries written to analyze the dataset.

## Project Objectives
The analysis focuses on uncovering trends within Netflix's content library. Key areas explored include:
  **Content Distribution:** Comparing the total number of movies vs. TV shows.
   **Categorization:** Analyzing content based on ratings (PG-13, R, TV-MA, etc.) and genres.
   **Release Trends:** Tracking how content additions have grown over the years and identifying top-producing countries.
  **Director & Cast Insights:** Querying specific actors or directors to see their footprint on the platform.

## Tech Stack

   **SQL** (Data Extraction, Aggregations, Grouping, and Filtering)
  **Excel** (Initial data preview)
